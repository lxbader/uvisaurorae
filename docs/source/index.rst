Installation
============
You can install this package via pip using the command::

    pip install uvisaurorae

to get the current release. If you want the latest code with the newest features (and bugs), you can clone the package
code from github like so::

    git clone https://github.com/lxbader/uvisaurorae.git

and install it from source using::

    pip install <path_to_source>
